This plugin inserts the code snippet for Targeter App.
== Description ==
Targeter App is Behavioral Targeting framework. You can take custom action based on the triggers selected. Triggers can be country of origin of visitor, time spent on page, engagement with the site, liking page on Facebook, tweeting your link, etc.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload contents of the zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your client Id in Targeter App settings page
